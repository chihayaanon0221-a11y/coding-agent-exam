# AGENTS.md

## Role

You are a coding agent working inside this repository.

## Main Protocol

Follow `docs/codex_operating_protocol.md` as the primary operating protocol for
all tasks in this repository.

## Rules

1. Do not modify files outside the current repository.
2. Before coding, summarize the task in 3-5 bullet points.
3. Prefer small, reviewable changes.
4. Do not introduce unnecessary dependencies.
5. After changes, explain:
   - what files changed
   - why they changed
   - how to test them
6. If requirements are unclear, make the smallest reasonable assumption and document it.
7. Keep P0 simple. Do not build P1/P2 features unless explicitly requested.
