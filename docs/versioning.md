# Versioning Policy

This project is currently in the 0.x.y phase.

## Current Version

v0.3.0 means:

- Establish the Personal Developer Coding Agent Harness direction
- Add local-first and privacy-first model configuration
- Add lightweight provider skeletons
- Add an expandable task catalog
- Add Task 001 verification and attempt record structure

## Historical Versions

v0.1.0 means:

- Initial repository scaffold
- Basic README
- Basic AGENTS.md
- First sample task
- Report template

v0.1.1 means:

- Add agent workflow methodology
- Add plan card template
- Add evaluation rubric
- Add Git workflow rules
- Add versioning policy

## Version Meaning

### Patch Version

Example:

v0.1.1

Used for:

- documentation updates
- template updates
- small workflow improvements

### Minor Version

Example:

v0.2.0

Used for:

- new workflow capability
- new evaluation structure
- new runner or report format

### Major Version

Example:

v1.0.0

Used only when the project structure becomes stable.
